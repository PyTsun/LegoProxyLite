# LegoProxyLite
A Roblox Proxy for accessing Roblox APIs through HTTPService

[![Join LegoProxy Discord](https://cdn.discordapp.com/attachments/1056074242325741578/1056131769977544735/image.png)](https://discord.gg/SnmVQ4NSTz)

## What's the difference?
LegoProxyLite works pretty much the same as LegoProxy but it lacks several features that the original has. however, LegoProxyLite is like a "straight to the point" Proxy unlike LegoProxy which the Proxy checks for conditions until the information is returned back.
